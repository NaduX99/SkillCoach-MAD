# Identity & News Feature Branch

This branch (`iamlasantha/identity-and-news`) implements the core user profile management and news insights systems.

## Features Implemented

### 1. User Identity & Profile
- **Profile Setup Flow**: A 5-step interactive onboarding process to capture user goals and experience.
- **User Profile Screen**: A polished dashboard displaying user statistics, achievements, and settings.
- **State Management**: Robust implementation using Riverpod `StateNotifierProvider` for persistent user data.
- **Aesthetics**: Premium glassmorphism design with smooth fade-in and slide-up animations.

### 2. World News & AI Insights
- **News Feed**: A dynamic list of curated AI and educational news articles.
- **Pull-to-Refresh**: Manual refresh logic to update the news stream.
- **Interactive Cards**: Category-tagged news cards with external link integration.
- **Shimmer Loading**: Simulated loading placeholders for enhanced perceived performance.

## Technical Highlights
- **Architecture**: Clean separation of concerns (Models, Providers, Widgets, Screens).
- **Quality Assurance**: Widget tests for core profile components.
- **Documentation**: Comprehensive READMEs for each sub-feature.

## Final Integration
All components are unified under a consistent design language and state management architecture.
